opengl.Texture

  • Home
  • opengl
  • Texture

Texture can be used to upload buffers of data to the GPU as well as allocate new buffers that reside solely on the GPU for offscreen rendering. Additionally, data can be read from the GPU using a fast-path readback based on Pixel Buffer Objects by calling Texture:toarray().
