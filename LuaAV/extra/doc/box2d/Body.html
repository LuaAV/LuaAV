Body:box(size, center, [angle])

A box can be created with the parameters below or a table of parameters. Valid parameters are center, size, angle, friction, restitution, density, isSensor. Size is a required parameter.


 -- the following are equivalent:
 body1:box({1, 1}, {0, 0})
 body2:box{
 size = {1, 1},
 center = {0, 0},
 }

Parameters:

size The size of the box
center The center of the box
[angle] The box angle
Body:circle(size, center, [angle])

A circle can be created with the parameters below or a table of parameters. Valid parameters are center, radius, angle, friction, restitution, density, isSensor. Radius is a required parameter.


 -- the following are equivalent:
 body1:circle(3)
 body2:circle{
 radius = 3
 }

Parameters:

size The size of the box
center The center of the box
[angle] The box angle

Body.mass GETSET

Mass can be set with either a numerical value or a table. If the value is a table, it can have parameters mass and center.


 body.mass = {
 mass = 1.,
 center = {0.1, 0},
 }
 -- or simply
 body.mass = 1.
