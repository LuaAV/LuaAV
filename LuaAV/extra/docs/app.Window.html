Detailed Description

An OpenGL Window. Manages OS event callbacks within the window and during window focus.

Current callbacks are: 

     Window:create()
     Window:init()
     Window:closing()
     Window:key(event, key)
     Window:modifiers()
     Window:mouse(event, btn, x, y, nclicks)
     Window:scroll(x, y, dx, dy)
     Window:resize()
     Window:draw()
