opengl.Mesh

  • Home
  • opengl
  • Mesh

Mesh efficiently draws large buffers of geometry data with the constraint that all geometry buffers except for indices must have the same number of cells.

Functions

Mesh:color(color) Set the color data
Mesh:draw() Draw the mesh
Mesh:index(index) Set the index data
Mesh:normal(normal) Set the normal data
Mesh.primitive GETSET The drawing primitive
Mesh:texcoord(texcoord) Set the texture coordinate data
Mesh:vertex(vertex) Set the vertex data

Functions

Mesh:color(color)

Color data must have 4 components of type Float32.

Parameters:

color Array of color data
Mesh:draw()

Draw the mesh

Mesh:index(index)

Indices must have 1 component of type UInt32

Parameters:

index Array of index data
Mesh:normal(normal)

Normal data must have 3 components of type Float32.

Parameters:

normal Array of normal data
Mesh.primitive GETSET

The drawing primitive used by the mesh. Valid values are GL_POINTS, GL_LINES, GL_LINE_STRIP, GL_LINE_LOOP, GL_TRIANGLES, GL_QUADS, and GL_POLYGON.

Mesh:texcoord(texcoord)

Texture coordinate data must have 2 or 3 components of type Float32.

Parameters:

texcoord Array of texture coordinate data
Mesh:vertex(vertex)

Vertex data must have 3 components of type Float32.

Parameters:

vertex Array of vertex data
