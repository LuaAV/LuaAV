Label(init)

Constructor

Parameters:

init Constructor properties
init.alignment The text alignment (Label.LEFT, Label.CENTER (default), Label.RIGHT)
init.bg draws a background (default not draw)
init.bgcolor sets background color (default is black)
init.color sets color (default is white)
init.ctx The label's context name
init.fontfile The file to load the font from
init.margin The x/y margin around the font in pixels (default {0, 0})
init.size The font size (default 8)
