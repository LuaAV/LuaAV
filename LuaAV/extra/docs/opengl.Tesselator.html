Detailed Description

The Tesselator module translates complex 2D shape outlines into triangles, filling in the shapes. It handles both concave and convex shapes.

  local tess = Tesselator()
  tess:BeginPolygon()
 
  -- first contour
  tess:BeginContour()
     tess:Vertex(0, 0)
     tess:Vertex(1, 0)
     tess:Vertex(1, 1)
     tess:Vertex(0.5, 1.5)
     tess:Vertex(0, 1)
  tess:EndContour()
 
  -- second contour
  tess:BeginContour()
     tess:Vertex(0.25, 0.25)
     tess:Vertex(0.75, 0.25)
     tess:Vertex(0.75, 0.75)
     tess:Vertex(0.25, 0.75)
  tess:EndContour()
 
  tess:EndPolygon()

Tesselator:geometry ( vertices,
indices
)

The geometry function extracts the triangle information from the tesselated contours. It fills an array of vertices and an array of indices with the tesselation data, which can be used with a Mesh to draw it to screen.

  local vertex = Array()
  local index = Array()
 
  tess:geometry(vertex, index)
 
  local mesh = Mesh(ctx)
  mesh:vertex(vertex)
  mesh:index(index)
  mesh.primitive = GL.TRIANGLES

Parameters:
verticesThe vertex array
indicesThe indices array
