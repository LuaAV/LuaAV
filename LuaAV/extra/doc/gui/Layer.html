gui.Layer

  • Home
  • gui
  • Layer

Within a gui.context, there are several layers for distinguising between different classes of widgets. Layers are typically overlaid on top of each other with the context testing them in order of priority for whether widgets in any particular layer receive an event. For example popup menus are usually high priority and would be the top-most layer with standard widgets getting events only if something in the popup layer doesn't take it.
