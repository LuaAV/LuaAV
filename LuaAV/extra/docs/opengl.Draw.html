Detailed Description

Draw is a high-level graphics interface for 2D drawing. It has a number of modes for filling and stroking common 2D shapes and curves that includes line cap and join styles.

Attributes

Draw.bezierDetail GETSET

Bezier detail

Draw.curveDetail GETSET

Curve detail

Draw.curveTightness GETSET

Curve tightness

Draw.ellipseMode GETSET

Possible values are Draw.CENTER, Draw.RADIUS, Draw.CORNER, Draw.CORNERS

Draw.fill GETSET

Sets whether or not to fill shapes

Draw.rectMode GETSET

Possible values are Draw.CENTER, Draw.RADIUS, Draw.CORNER, Draw.CORNERS

Draw.stroke GETSET

Sets whether or not to stroke shapes

Draw.strokeCap GETSET

Possible values are Draw.SQUARE, Draw.PROJECT, Draw.ROUND

Draw.strokeJoin GETSET

Possible values are Draw.MITER, Draw.BEVEL, Draw.ROUND

Draw.strokeWeight GETSET

The weight of a stroked path.

Draw:colorMode GETSET

Possible values are Draw.RGB and draw.HSB

Draw:colorRange GETSET

The color range

Draw:fillColor GETSET

Fill color

Draw:strokeColor GETSET

Stroke color
