  • Slab

Shaders are specified as .shl files, which are a Lua-based file format for describing shader code and parameters. Shader uniforms can be set via the Shader:param() method. Shader attributes are accessed through internal ShaderAttribute objects that can be retrieved by name from the Shader:attr() method.
