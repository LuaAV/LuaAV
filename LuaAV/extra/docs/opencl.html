Detailed Description

The OpenCL module contains both high-level and low-level bindings. The low-level bindings provide wrappers around OpenCL API functions while the high-level functions enable the use of OpenCL with minimal amounts of code. With high-level functions the OpenCL context and its resources are automatically managed.
