Send:send(message, ...)

-- list form: sender:send("/address", 1, 2., "three", true)

-- table form: sender:send{ addr="/address", 1, 2., "three", true }

-- bundle example (Timetags are not supported yet.) sender:send{ { addr="/address1", 1, 2., }, -- nested bundle: { { addr="/address2", "three", }, { addr="/address3", true, }, } }

Parameters:

message or bundle (table) address (string OSC address)
... (numbers, strings, booleans, nils) arguments for single message

Return:

  • Number of bytes sent
