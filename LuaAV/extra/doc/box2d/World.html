A world is the physics space in which all bodies and shapes are created in. It runs the physics and dynamics. Callbacks can be attached to a world in order to manage collisions and other events.

Functions

World:body(pos, angle, [type])

Bodies can be created with the listed parameters or by passing in a table with all of the body parameters. The list of paramters is: position, type, angle, linearVelocity, angularVelocity, linearDamping, angularDamping, allowSleep, awake, fixedRotation, active, gravityScale. Only position is required.


 -- The following are equivalent:
 local body1 = world:body({0, 0}, 0, World.DYNAMIC)
 local body2 = world:body{
 position = {0, 0},
 angle = 0,
 type = World.DYNAMIC,
 }

Parameters:

pos The position of the body
angle The angle of the body
[type] The body type, default DYNAMIC

Return:

  • The body
World.contactEvents GETSET

The value is a bitfield whose possible bits are World.BEGIN_CONTACT, World.END_CONTACT, World.PRE_SOLVE, World.POST_SOLVE


 -- list for pre- and post-solve contact events
 world.contextEvents = World.PRE_SOLVE + World.POST_SOLVE
 -- PreSolve callback
 function world:PreSolve(contact)
 -- do something (never use contact outside of this function)
 end
 -- PostSolve callback
 function world:PostSolve(contact)
 -- do something (never use contact outside of this function)
 end

World:distanceJoint(body1, body2, [t])

Joint parameters are collideConnected, frequencyHz, dampingRatio


 local distj = world:distanceJoint(b1, b2, {
 collideConnected = true,
 })

Parameters:

body1 The first body
body2 The second body
[t] The optional table of joint parameters

Return:

  • The distance joint

World:revoluteJoint(body1, body2, [t])

Joint parameters are collideConnected, lowerAngle, upperAngle, maxMotorTorque, motorSpeed, enableLimit, enableMotor


 -- an angle limited revolute joint
 local revj = world:revoluteJoint(b1, b2, {
 collideConnected = false,
 lowerAngle = pi/2,
 upperAngle = 3*pi/2
 })

Parameters:

body1 The first body
body2 The second body
[t] The optional table of joint parameters

Return:

  • The revolute joint
